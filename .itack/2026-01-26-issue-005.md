# Add ccx send command

**Problem:** We had to use `tmux send-keys -t ccx-5a358df8 Enter` directly to send input to a session. Users shouldn't need to know ccx uses tmux internally.

**Solution:** Add a `ccx send <session> <message>` command that wraps tmux send-keys.

```rust
// In tmux.rs
pub fn send_keys(session: &str, text: &str) -> Result<()> {
    Command::new("tmux")
        .args(["send-keys", "-t", session, text, "Enter"])
        .status()?;
    Ok(())
}

// New CLI command
Commands::Send { session, message } => cmd_send(&session, &message),
```

**Files:** `src/main.rs`, `src/tmux.rs`
